# Steps to Run
First clone the project :__
  git clone https://link-to-project__
Go to the project Directory__
   cd my_project__
Install the Dependencies
  pip install flask
  pip install flask-wtf
  pip install selenium
  pip install beautifulsoup4
Start the server
     flask --app app run
